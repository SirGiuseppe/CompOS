Welcome to the CompOS Project

The CompOS project is the journey to create an excellent DE made by everyone, for everyone. This includes support for apps by other vendors, skinning, and (yes) custom distributions (if you wanna make it sound lame, it is basically Linux on Windows) But, if it was truly Linux on Windows, we wouldn't need this page. The showstopper(s!) of the project follows:

The first in the industry.

CompOS is the first VBDE to be publicly maintained and developed. That means that people from all skill levels can come along and help.

Don't code in VB.NET? No problem!

This is for all coding languages, just grab yourself a clone of the git repo and get coding!

New release every week!

We hand-pick a release version every week. That means you get a whole new set of features to test every week, bugs and all! We try to pick the most stable releases and test them before packaging them in a fancy installer and releasing them officially.

Multiple app vendors: Unlike other VB.NET-based DE's, we aim to create an open market with user-created apps, as well as other apps by vendors like you guys.

[And of course!] Flavoring: Create customizable skins to suit what you want out of your PC.

So come on down and get coding!
